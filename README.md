# AstroNvim Template

## Keyboard Shortcuts


- **Movimentação com o Cursor:**
  - Use `Alt` + setas para se movimentar.

- **Visualização de Tipos e Definições:**
  - Pressione `K` para ver o tipo da variável, definições, etc.

- ** Code action menu**
 Use gd para ver o menu de acionamento de código.
